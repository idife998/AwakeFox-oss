# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in AwakeFox, please report it responsibly by emailing:

**hello@justfox.dev**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes (optional)

## Response Time

We aim to respond to security reports within **48 hours** and will work to address valid vulnerabilities as quickly as possible.

## Scope

This security policy applies to:
- The AwakeFox executable
- Source code in this repository

## Out of Scope

- Issues in third-party software or operating systems
- Social engineering attacks
- Physical attacks

## Disclosure

We kindly ask that you:
- Give us reasonable time to fix the issue before public disclosure
- Do not exploit the vulnerability beyond what is necessary to demonstrate it
